About VATSIM

Before we start how to find your way around VATSIM and what to do, let's clarify a few questions

What is VATSIM?​

VATSIM is an online aviation network where all kinds of people are on the road. From real pilots and pilots to absolute beginners VATSIM has everything to offer.

You can start relatively easy after your registration and under the condition that you have the pilot clients and your flightism installed (in the true sense of the word).

The first steps as a pilot​

  1. First of all you have to register on VATSIM.net.

  2. After you have created an account you can download the appropriate pilot client depending on the simulator you have installed.

    • P3D, MSFS vPilot
    • XP11, XP12 xPilot

    Click here to go to the appropriate page to get more information about it

info

To see other aircraft in your simulator you need a suitable model matching for Xplane I can recommend "Bluebell CSL". For the MSFS "FSTL Live Traffic".

First steps as ATCo​

Example of the vACC Germany.

The step to the ATCo is a bit more complex and also more difficult. First you have to register with VATSIM in a region and then you have to assign yourself to a vACC in this region. This is incredibly complex, so no detailed description here.

If you have overcome these hurdles you can assign yourself to a RG (Reginoal Group) on the website of vACC Germany.

After that you can register in the appropriate forum of the RG for the Basic Lesson or for the newly introduced S1 Infosession and start to pilot after a Moodle course.

caution

The description is not 100% consistent with the correct vACC's. Also, this path is extremely simplified, so for the vACC Germany I can only refer you to this page where you can get more info.
